About MindGlow

Mindglow is a set of reusable blocks for Pinegrow editor. It allows an easy creation of unique, responsive websites with just a little effort. It is based on Bootstrap 4 framework and comes with its own components and stylesheets. Download and enjoy!

Features

Bootstrap 4.0

MindGlow is based on a modern, responsive framework Bootstrap 4

Pinegrow 4.3

MindGlow is created from scratch in Pinegrow

Font Awesome 5.0

MindGlow utilizes the latest version of Font Awesome
